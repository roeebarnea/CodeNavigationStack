# Change Log

All notable changes to the Code Navigation Stack extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.9] - 2024

### Added

- Code Navigation Tree view in Explorer sidebar
- Code Navigation Stack panel for tracking current path
- Interactive navigation by clicking on tree nodes or stack entries
- Color-coded file indicators for better visual distinction
- Automatic tracking of cursor position in functions, methods, and properties

### Changed

- Improved code organization and documentation
- Enhanced type safety throughout the codebase
- Better error handling in core functions
- Optimized performance for large navigation trees

### Fixed

- Improved stability of navigation tracking
- Better handling of edge cases in symbol detection

## [Unreleased]

### Planned

- Configuration options for color schemes
- Keyboard shortcuts for quick navigation
- Export/import navigation history
- Search functionality within navigation tree
